ArcYou HTML Demo 문서

이 파일은 Marker + ArcYou 전처리 파이프라인의 HTML 처리 데모용 문서입니다.

1. 목표

ArcYou는 사용자가 업로드한 다양한 형식의 문서를 마크다운 + 레이아웃 + 임베딩 구조로 변환하여 개인 지식 베이스로 저장하는 것을 목표로 합니다.

2. 지원 포맷 (데모 기준)

  • PDF (예: test.pdf)
  • 이미지 (예: test.png)
  • HTML (현재 문서)

3. RAG 관점에서의 활용

전처리된 마크다운은 LangChain 기반 RAG 에이전트에서 DocumentChunk 단위로 검색 및 재조합되어, 사용자의 질문에 대한 맞춤형 답변을 생성하는 데 사용됩니다.
